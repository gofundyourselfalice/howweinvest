Save each file into this folder (`images/logos/`) using the **filename** in the
first column (`.svg` ideal, transparent `.png` fine). The page then uses it
automatically.

## Notes
- Filenames must match exactly (lowercase, hyphens as shown). Either `.svg` or `.png` works.
- After dropping files in, refresh the page — no code changes needed.
- `.svg` from Brandfetch/press pages is sharpest; the direct Clearbit images are `.png`.
