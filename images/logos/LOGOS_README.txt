HOW TO ADD REAL PLATFORM LOGOS
==============================

Drop each provider's logo into THIS folder (images/logos/) using the exact
filename below. SVG is best; a transparent PNG also works. As soon as a file
is here, the "Where you invest" page uses it automatically. If a file is
missing, the page falls back to a live logo service, then to initials.

TIPS
- Square or wide logos both work; they're scaled to fit the logo box.
- Most companies have a "Press", "Media", "Brand" or "Newsroom" page with
  downloadable logo assets. That's the cleanest, properly-licensed source.
- After adding files, just refresh the page. No code changes needed.
